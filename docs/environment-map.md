# 环境分工

## 本机

本机适合做开发、数据下载、策略检查、回测观察和报告整理。

- 迅投终端安装在：`D:\迅投极速交易终端睿智融科版`
- 桌面快捷方式指向：`D:\迅投极速交易终端睿智融科版\bin.x64\XtItClient.exe`
- XtData 本地服务由迅投客户端提供，终端打开并登录后，外部 Python 才能稳定读取数据。
- 本项目目录：`D:\wh_xz\quant`

## 远程服务器

远程服务器适合做长时间运行、隔离部署、云端值守或比赛环境复现。已确认能通过 RDP 登录，是干净 Windows 桌面环境。

敏感连接资料放在 `private\远程服务器资源.md`，不要外发，也不要提交到仓库。

## 两者差异

- 本机已经有迅投终端、数据缓存和当前策略文件，适合快速调试。
- 远程服务器需要重新安装终端、同步策略、下载数据或复制必要数据，适合长期运行。
- 本机数据与客户端 UI 显示一致；远程服务器如果重新下载，必须再次抽检数据范围和最后交易日。
- 策略在本机能跑，不代表远程服务器已经具备相同数据、账号、权限和终端状态。

## 安全边界

- 回测版本、模拟盘版本和实盘版本必须分开管理。
- 当前项目工具默认只读，不会调用 `passorder`、`order_shares` 或真实下单动作。
- 涉及模拟盘或实盘前，需要确认账户、交易模式、下单 API、终端界面状态和撤单风控。
